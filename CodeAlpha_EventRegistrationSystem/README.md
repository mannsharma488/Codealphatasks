# Event Registration System

## Project Description

This is an Event Registration System developed using Django. It allows users to view events and register by submitting their details.

## Features

- View Events
- Register for Events
- Store Registration Details
- Django Admin Panel
- User-friendly Interface

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS

## Installation

1. Install the required packages:

```bash
pip install -r requirements.txt
```

2. Apply migrations:

```bash
python manage.py migrate
```

3. Run the server:

```bash
python manage.py runserver
```

4. Open your browser:

```
http://127.0.0.1:8000
```
