# URL Shortener

## Project Description
This is a URL Shortener web application built using Flask and SQLite. It converts long URLs into short and easy-to-share links.

## Features
- Shorten long URLs
- Custom Short Code
- URL Validation
- Automatic Redirect
- Click Counter
- Analytics API
- Copy Short URL
- Responsive User Interface

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Installation

1. Install the required package:

```bash
pip install -r requirements.txt
```

2. Run the project:

```bash
python app.py
```

3. Open in your browser:

```
http://127.0.0.1:5000
```
